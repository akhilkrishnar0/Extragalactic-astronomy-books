# Extragalactic-astronomy-books

This repository is a curated academic reading guide for galaxy structure,
dynamics, morphology, nearby galaxies, low-surface-brightness galaxies,
and observational galaxy evolution.

This repository does not host copyrighted textbook PDFs. Commercial books
are listed only with bibliographic metadata, DOI/publisher links, and legal
access routes. Open-access PDFs are included or linked only when redistribution
is permitted by the publisher or author.
